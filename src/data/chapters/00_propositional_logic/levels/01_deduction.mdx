<Speak speaker="sapphire">
<LocalizedMarkdown locale="en">
You can think so, but you're talking about the **semantics** of a string. While it is good to know the **semantics** of every string if you're not  a robot construct, the **semantics** cannot be used at all when doing formal proofs. Only **syntactics** can be used, which are the formal structure of the string and the deduction rules. 
</LocalizedMarkdown>
<LocalizedMarkdown locale="zh">
你可以这么想，但你正在讨论的是字符串的**语义**。虽然如果你不是机器人构装体的话，知道字符串的**语义**会很有帮助，但在做形式化证明的时候**语义**完全没法使用。你只能用**语法**，也就是形式化的字符串结构和推理规则。
</LocalizedMarkdown>
</Speak>
